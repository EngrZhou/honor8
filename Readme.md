## honor8
华为荣耀8全网通刷入第三方ROM后会导致电信卡通话无声音，因为语音通话使用2G/3G网络，荣耀8对对电信2G/3G（CDMA/CDMA2000）的支持采用的是外挂基带芯片，可能导致了Open Kirin的不支持。若要使用4G网络通话需开启VoLTE，但Open Kirin官方没打算支持VoLTE（貌似荣耀8也不支持VoLTE）。
